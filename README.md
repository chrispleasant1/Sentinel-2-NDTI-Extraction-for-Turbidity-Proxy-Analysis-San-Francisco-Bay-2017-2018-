# Sentinel-2-NDTI-Extraction-for-Turbidity-Proxy-Analysis-San-Francisco-Bay-2017-2018-
Google Earth Engine script extracting Sentinel-2 NDTI values across San Francisco Bay (2017-2018). Includes cloud masking, turbidity index calculation, median compositing, and GeoTIFF export. Developed for master's research on satellite-derived turbidity as a microplastic proxy.
